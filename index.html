Starium, le jeu

Starium est la version moderne d'un jeu datant des années 1970 lorsque les ordinateurs tenaient dans des pièces entières.

La toute première version du jeu a été écrite par Mike Mayfield en Basic. Il s'agissait d'un amusement entre programmeur mais le succès a été tel qu'il a été installé sur pratiquement tous les ordinateurs de l'époque.


But du jeu

Vous êtes le capitaine d'un vaisseau et vous devez pacifier la zone.

Protégez votre vaisseau, noté E comme Endurci sur l'écran.
Vous devez neutraliser vos ennemis, les Kipicks notés K.
Pour cela, vous disposez de lasers, de torpilles et de bases de ravitaillement.


Un jeu au clavier

Le jeu se joue au clavier. Des boutons raccourcis ont été mis à l'écran pour voir les commandes disponibles, mais il est nettement plus facile d'utiliser le clavier.

Note

La meilleure expérience de jeu se fait sur ordinateur.
Le jeu n'a pas été conçu pour les téléphones. Il peut fonctionner de manière correcte sur les tablettes.

Bon amusement !
